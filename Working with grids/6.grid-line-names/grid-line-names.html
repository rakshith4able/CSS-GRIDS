grid-line-names

        grid-template-columns and grid-template-rows performs 2 functions:
        1)To define track sizing by specifying width 
        2)defines the line names created by grid columns or grid rows

        Syntax is as follows:

        grid-template-columns:[linename] size;

        grid-template-rows:[linename] size;

        //line name can be anything we want
